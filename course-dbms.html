Database Management

Course Overview

This course provides a detailed understanding of database design, normalization, SQL queries, and working with both relational and non-relational databases.

Course Modules

  • Module 1: Introduction to Databases
  • Module 2: Relational Database Design
  • Module 3: SQL Query Writing
  • Module 4: Advanced SQL & Stored Procedures
  • Module 5: NoSQL Databases (MongoDB, Cassandra)
